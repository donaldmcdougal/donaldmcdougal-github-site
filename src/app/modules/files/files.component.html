Files Provides file operation utilities in Java.

The file utilities provide the following:

  • Copying a directory and all of its contents.
  • Deleting a directory and all of its contents.
  • Splitting a file into pieces. Useful for very large files.
  • Assembling split file pieces back into a file.
  • Getting a file extension.
  • Hiding and unhiding files. Works in Windows and Linux. Not tested in MacOS, but probably works.
  • Listing files recursively.
  • Writing a string to file.
